# AI 驱动量化平台：Numerai、Boosted.ai、Kensho 深度分析

> 日期：2026-03-14
> 这三个平台代表了 AI 在量化投资中三种截然不同的应用路径

---

## 一、Numerai：众包量化的颠覆者

### 1.1 背景

- 2015 年创立于旧金山
- 2025 年 11 月完成 **3000 万美元 C 轮**融资，估值 **5 亿美元**（较 2023 年增长 5 倍）
- 2025 年获得 **JPMorgan 5 亿美元容量承诺**
- 2024 年净收益 **25.45%**，Sharpe Ratio **2.75**

### 1.2 核心机制

```
┌─────────────────────────────────────────────────────────┐
│                   Numerai 运作机制                        │
│                                                         │
│  全球数据科学家（数千人）                                   │
│       │ 下载加密数据集（特征已脱敏，不知道是哪只股票）          │
│       │ 训练 ML 模型                                      │
│       │ 提交每周预测（质押 NMR 代币）                        │
│       ▼                                                 │
│  Numerai Meta-Model                                     │
│       │ 聚合所有模型的预测                                  │
│       │ 按相关性加权（鼓励多样性）                           │
│       ▼                                                 │
│  交易全球股票（市场中性策略）                                │
│       │                                                 │
│       ▼                                                 │
│  收益分配：准确预测者获得 NMR 奖励，不准确者损失质押           │
└─────────────────────────────────────────────────────────┘
```

### 1.3 核心创新点

**数据加密（防过拟合的关键）**：
- 特征数据完全脱敏，数据科学家无法知道自己在预测哪只股票
- 这迫使模型学习真正的统计规律，而非记忆特定股票的历史
- 有效防止了传统量化研究中的"数据挖掘偏差"

**Meta-Model 聚合**：
- 聚合数千个独立模型，多样性带来稳健性
- 按相关性加权：与其他模型相关性低的模型获得更高权重（鼓励差异化）
- 类似于集成学习（Ensemble Learning）的极致版本

**NMR 代币激励**：
- 数据科学家质押 NMR 代币，预测准确则获奖，不准确则损失质押
- 将激励机制与预测质量直接挂钩，筛选出真正有效的模型

### 1.4 费率结构

- 管理费：**1%**（传统对冲基金通常 2%）
- 绩效费：**20%**（与传统相同）
- 整体成本低于传统量化对冲基金

### 1.5 对传统量化机构的启示

| 传统量化 | Numerai 模式 |
|---------|-------------|
| 内部团队，模型集中 | 全球众包，模型多样 |
| 研究员知道数据含义 | 数据脱敏，防止过拟合 |
| 单一 Alpha 来源 | Meta-Model 聚合多元 Alpha |
| 人才竞争激烈 | 开放竞赛，降低人才门槛 |

**Fidelity 可借鉴**：建立内部"模型竞赛"机制，让不同团队提交预测，通过 Meta-Model 聚合，提升整体预测稳健性。

## 二、Boosted.ai（Alfa 平台）：Agentic AI 投研助手

### 2.1 背景

- 2017 年创立
- 2024 年 11 月完成 **1500 万美元**融资
- 服务 **300+ 客户**，管理资产超 **3 万亿美元**
- 客户包括资产管理公司、财富管理机构、对冲基金

### 2.2 核心产品：Alfa（2025 年升级版）

**Alfa 是一个 Agentic AI 平台**，核心理念是"镜像真实投资工作流，自动化重复性工作"：

```
传统分析师工作流（40 小时）：
  手动收集数据 → 建 Excel 模型 → 写研究报告 → 更新监控

Alfa 自动化后（20 分钟）：
  AI 自动监控 → AI 分析 → AI 生成报告草稿 → 分析师审核
```

**核心功能模块**：

| 功能 | 描述 |
|------|------|
| AI 模型构建 | PM 输入选股逻辑和重要变量，AI 结合大数据构建回测模型 |
| 因子时序分析 | 比较不同因子分位组的 AI 预测，识别因子轮动机会 |
| 自动监控 | 持续监控持仓和市场，异常时主动推送 |
| ETF 分析 | AI 解析 ETF 持仓，评估风险指标和费率 |
| 报告生成 | 自动生成投资委员会和客户信任的分析报告 |

### 2.3 技术架构（推断）

基于公开信息，Alfa 的技术架构应包含：
- **Agentic RAG**：多步骤检索和推理，处理复杂投研问题
- **多数据源集成**：与主流数据合作伙伴（Bloomberg、FactSet 等）集成
- **ML 因子模型**：基于 PM 输入的变量，自动训练和回测因子模型
- **自然语言界面**：PM 用自然语言描述投资逻辑，AI 转化为量化模型

### 2.4 核心价值主张

**降低量化门槛**：让没有量化背景的 PM 也能使用 AI 构建和验证投资逻辑，不需要写代码。

## 三、Kensho（S&P Global AI 部门）：金融 NLP 基础设施

### 3.1 背景

- 2013 年哈佛创立，创始人 Daniel Nadler（哈佛 PhD）
- 2018 年被 S&P Global 以 **5.5 亿美元**收购
- 现为 S&P Global 的 AI 创新中心，为 Capital IQ Pro、Ratings 等产品注入 AI 能力

### 3.2 核心产品

**Kensho Classify（2023 年发布）**：
- 基于"小型语言模型"的主题标签工具
- 零样本（Zero-Shot）分类：无需训练即可标注新类别
- 预训练行业分类（GICS 等），开箱即用
- 用途：文档自动分类、新闻标注、研究报告主题标签

**S&P AI Benchmarks by Kensho（2024 年发布）**：
- 评估 LLM 在金融领域的能力
- 三个维度：定量推理、业务基本面理解、文档信息提取
- 已成为金融 LLM 评估的行业参考标准

**NLP 工具套件**：
- 语音转文字（财报电话会议记录）
- 文档结构化（PDF → 结构化数据）
- 实体识别（公司名、人名、财务数字）

### 3.3 战略价值

Kensho 的核心价值不仅在于产品本身，更在于：
1. **为 S&P Global 数据产品注入 AI 能力**：Capital IQ Pro 的 AI 功能大量来自 Kensho
2. **LLM 基准的行业影响力**：S&P AI Benchmarks 影响了整个行业对金融 LLM 的评估方式
3. **非结构化数据结构化**：将财报、新闻、电话会议记录转化为可量化的信号

## 四、三者对比与启示

| 维度 | Numerai | Boosted.ai | Kensho |
|------|---------|-----------|--------|
| 核心创新 | 众包 Meta-Model | Agentic AI 投研 | 金融 NLP 基础设施 |
| 目标用户 | 数据科学家 | 资产管理 PM | 金融数据平台 |
| AI 应用方式 | 聚合众包模型 | Agentic RAG + ML | NLP + 分类 |
| 商业模式 | 对冲基金 | SaaS 订阅 | 嵌入 S&P 产品 |
| 2024 年亮点 | 25% 净收益 | 1500 万融资 | LLM 基准发布 |

### 对 Fidelity 的综合建议

1. **借鉴 Numerai 的众包机制**：建立内部量化模型竞赛，聚合多团队 Alpha
2. **参考 Boosted.ai 的 Agentic 架构**：将 SuperSearch 升级为 Agentic RAG，支持复杂投研问题
3. **学习 Kensho 的 NLP 流水线**：建立内部财报/新闻 NLP 处理流水线，提取量化信号
