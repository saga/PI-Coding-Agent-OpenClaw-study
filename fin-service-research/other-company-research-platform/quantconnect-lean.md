# QuantConnect / LEAN：开源量化回测平台分析

> 日期：2026-03-14

---

## 一、平台概述

QuantConnect 是全球最大的开源量化交易平台，底层引擎 LEAN 在 GitHub 上有 **12,300+ Star**，**3,700+ Fork**，是量化开发者社区的核心基础设施。

**定位**：云端 IDE + 回测引擎 + 实盘交易，一站式量化研究和交易平台。

---

## 二、核心功能

### 2.1 研究环境

- Jupyter Notebook 集成，支持 Python 和 C#
- 直接访问历史数据（无需下载），在云端完成数据探索
- 与回测引擎共享同一套 API，研究代码可直接转为策略代码

### 2.2 回测引擎（LEAN）

```
LEAN 回测引擎特性：
├── 多资产支持：股票、期权、期货、外汇、加密货币、CFD
├── 点对点回测：内置费用、滑点、价差调整
├── 真实保证金模型：模拟真实的保证金要求和强平逻辑
├── 事件驱动架构：逐 Tick 或逐 Bar 模拟
└── 并行优化：云端同时运行数千次参数优化
```

### 2.3 实盘交易

支持 Interactive Brokers、Alpaca、Coinbase 等数十个券商和交易所直连，回测策略可一键部署到实盘。

### 2.4 数据覆盖

| 数据类型 | 覆盖范围 | 来源 |
|---------|---------|------|
| 美股 | 全市场，含退市股，日线/分钟/Tick | QuantConnect 自有 |
| 期权 | 美股期权链，含历史隐含波动率 | 付费订阅 |
| 期货 | 主要商品和金融期货 | 付费订阅 |
| 加密货币 | 主流交易所历史数据 | 免费 |
| 基本面数据 | 有限，需额外订阅 | Morningstar 等 |

## 四、LEAN CLI（本地部署）

对于机构用户，LEAN CLI 允许在本地或私有云运行完整的 LEAN 引擎：

```bash
# 安装
pip install lean

# 初始化项目
lean init

# 本地回测
lean backtest "My Strategy"

# 本地研究环境
lean research
```

**优势**：无供应商锁定，数据完全自控，可集成内部数据源。

## 五、与专业机构平台的差距

| 维度 | QuantConnect/LEAN | FactSet/Bloomberg |
|------|------------------|------------------|
| 基本面数据 | 有限 | 极为丰富 |
| 因子研究工具 | 基础 | 专业完善 |
| 风险模型 | 无内置 | 集成 Barra/Axioma |
| 合规/审计 | 弱 | 强 |
| 技术门槛 | 高（需编程） | 低（GUI 友好） |
| 成本 | 极低 | 极高 |
