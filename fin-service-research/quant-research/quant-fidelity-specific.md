## 一、Fidelity International 量化研究现状调研

### 1.1 Fidelity International 概况

| 维度 | 信息 |
|------|------|
| 管理规模 | 约 7000 亿美元 AUM（2024） |
| 总部 | 英国伦敦 |
| 主要市场 | 欧洲、亚太、新兴市场 |
| 投资风格 | 以主动管理为主，量化为辅 |
| 量化团队 | Systematic Investing 部门 |

**注**：Fidelity International 与 Fidelity Investments（美国）是独立的两家公司，本文聚焦 Fidelity International。

### 1.2 现有量化能力评估

**已有优势**：
- **基本面研究深度**：全球 400+ 分析师，覆盖 2500+ 公司，研究数据库积累深厚
- **专有数据资产**：分析师评级、公司访谈记录、内部预期数据
- **ESG 整合**：在 ESG 评分和整合方面有较强积累
- **亚太本地化**：在香港、上海、东京、新加坡有本地研究团队

**量化能力差距**（与顶级量化机构对比）：
- 因子库系统化程度不足，缺乏统一的因子管理平台
- 回测基础设施相对落后，Point-in-Time 数据库不完善
- MLOps 能力薄弱，模型从研究到生产的流程不规范
- 另类数据利用率低，采购和评估体系不成熟
- 量化与基本面研究的融合机制缺失

## 二、Fidelity 多资产量化策略

### 2.1 多资产量化的挑战

Fidelity International 管理大量多资产基金（目标日期基金、平衡型基金、多资产收益基金），量化方法在多资产场景面临独特挑战：

**数据异质性**：
- 股票：日频价格、季度财报、实时新闻
- 固定收益：收益率曲线、信用利差、久期
- 商品：期货曲线、库存数据、季节性
- 另类资产：私募估值滞后、流动性差

**因子跨资产迁移问题**：
- 股票因子（价值、动量、质量）在固收中的对应物不同
- 跨资产相关性在危机时期急剧上升（相关性崩溃）
- 不同资产类别的再平衡频率和交易成本差异巨大

### 2.2 股票 + 固定收益 + 多资产的统一因子框架

**统一因子框架设计原则**：

```
宏观因子层（跨资产通用）
├── 经济增长因子（GDP 预期、PMI）
├── 通胀因子（CPI、盈亏平衡通胀率）
├── 流动性因子（信用利差、VIX）
└── 货币政策因子（利率预期、央行资产负债表）

资产类别因子层
├── 股票：价值、动量、质量、低波动、规模
├── 固收：久期、信用质量、流动性溢价、凸性
├── 商品：期货曲线斜率（Carry）、动量、季节性
└── 货币：利差（Carry）、动量、估值（PPP）

特异性因子层（资产级别）
└── 个股/个券的特有因子
```

**Two Sigma Factor Lens 的参考价值**：
- 其正交化多资产因子模型是目前最成熟的公开框架之一
- Fidelity 可以在此基础上加入专有的基本面因子

### 2.3 跨资产类别的风险模型

**多资产风险模型架构**：

| 组件 | 方法 | 说明 |
|------|------|------|
| 股票风险 | Barra/Axioma 多因子模型 | 行业标准，直接采购 |
| 固收风险 | 久期 + 关键利率久期（KRD） | 利率曲线风险分解 |
| 信用风险 | 信用利差因子 + 违约概率 | IG/HY 分开建模 |
| 跨资产相关性 | DCC-GARCH 动态相关性模型 | 捕捉危机时期相关性变化 |
| 尾部风险 | CVaR + 历史情景压力测试 | 2008、2020 等极端情景 |

**相关性崩溃的处理**：
- 正常市场：使用历史相关性矩阵
- 压力情景：使用危机期间的相关性矩阵（股债相关性可能从 -0.3 变为 +0.5）
- 混合方法：Regime-Switching 模型，根据市场状态切换相关性矩阵

### 2.4 目标日期基金的量化优化

**目标日期基金（TDF）的量化挑战**：
- 随时间自动调整股债比例（Glide Path）
- 需要在长期收益和临近退休时的风险控制之间平衡
- 个人化需求：不同投资者的风险偏好、退休时间不同

**量化优化方向**：

**1. 动态 Glide Path 优化**：
```
传统方法：固定的股债比例下滑路径（如距退休 30 年 90% 股票 → 退休时 40%）
量化方法：基于宏观因子动态调整
  - 经济扩张期：适当提高股票比例
  - 衰退信号出现：提前降低风险
  - 利率上升期：缩短固收久期
```

**2. 个人化 Glide Path**：
- 输入：投资者年龄、风险承受能力、其他资产（房产、养老金）
- 输出：个性化的资产配置路径
- 技术：强化学习（RL）优化长期效用函数

**3. 再平衡优化**：
- 传统：固定频率（季度）再平衡
- 量化：基于偏离度触发（当实际配置偏离目标超过阈值时再平衡）
- 考虑交易成本和税务影响

## 三、Fidelity 亚太市场量化挑战

### 3.1 中国 A 股市场特殊性

**制度性约束**：

| 特殊规则 | 内容 | 量化影响 |
|---------|------|---------|
| 涨跌停板 | ±10%（ST 股 ±5%） | 价格不连续，动量因子失效 |
| T+1 交易 | 当日买入次日才能卖出 | 无法日内平仓，增加隔夜风险 |
| 退市制度 | 连续亏损可能退市 | 幸存者偏差更严重 |
| 北向资金 | 外资通过沪深港通流入 | 外资流向是重要另类因子 |
| 融券限制 | 做空成本高，标的有限 | 多空策略受限，因子多头效应更重要 |
| 印花税 | 卖出征收 0.1% | 高换手策略成本显著增加 |

**A 股特有因子**：
- **北向资金因子**：外资净买入作为聪明钱信号
- **融资余额因子**：融资买入反映散户情绪（反向指标）
- **龙虎榜因子**：大宗交易和机构席位数据
- **分析师覆盖变化**：新增覆盖/评级上调的信息含量
- **限售股解禁**：大股东减持压力因子

**回测注意事项**：
- 必须使用 A 股专用的 Point-in-Time 数据库（Wind、聚宽）
- 涨跌停处理：被封板的股票无法成交，需要特殊处理
- 退市股票必须纳入回测（幸存者偏差在 A 股尤为严重）
- 交易成本模型需包含印花税、佣金、冲击成本

### 3.2 亚太其他市场特殊性

**日本市场**：
- 交叉持股（Cross-shareholding）导致价值因子长期失效
- 2023 年东京证交所改革推动价值回归，创造历史性机会
- 日元汇率对外资收益影响显著，需要货币对冲决策

**韩国市场**：
- 财阀（Chaebol）结构导致公司治理折价
- 外资持股限制影响流动性
- 2024 年"企业价值提升计划"推动价值因子复苏

**印度市场**：
- 高增长但估值贵，传统价值因子效果差
- 散户占比高，动量因子有效
- 数据质量和覆盖率相对较好（NSE/BSE 数据完整）

**东南亚市场**：
- 流动性差，大市值股票有限
- 数据质量参差不齐
- 适合主题投资而非系统化量化

### 3.3 亚太数据质量和覆盖率问题

**主要数据挑战**：

| 问题 | 影响 | 解决方案 |
|------|------|---------|
| 财报披露不规范 | 财务因子计算困难 | 使用 Wind/Bloomberg 标准化数据 |
| 历史数据短 | 回测样本不足 | 结合模拟数据，谨慎解读结果 |
| 另类数据覆盖差 | 信号来源有限 | 优先采购亚太专项另类数据 |
| 多语言文本 | NLP 模型效果差 | 需要中文/日文/韩文专用模型 |
| 时区差异 | 数据对齐复杂 | 统一使用 UTC 时间戳 |

### 3.4 多语言研究文档处理

**Fidelity 亚太研究的语言挑战**：
- 中文：A 股公司公告、中文研究报告
- 日文：日本公司 IR 文件、日文新闻
- 韩文：韩国财报、韩文分析师报告

**技术方案**：

**方案一：多语言 LLM**
- 使用 GPT-4/Claude 的多语言能力
- 优点：开箱即用，覆盖所有语言
- 缺点：金融专业术语理解不够精准

**方案二：语言专用模型**
- 中文：FinBERT-Chinese、ERNIE-Finance
- 日文：Japanese FinBERT
- 优点：专业术语理解好
- 缺点：需要分别维护多个模型

**方案三：翻译 + 英文模型**
- 先用翻译 API（DeepL/Google）翻译成英文
- 再用英文金融 LLM 处理
- 优点：简单，利用成熟英文模型
- 缺点：翻译损失信息，成本较高

**建议**：短期用方案三快速上线，中期建设方案二的中文专用模型（A 股研究优先）。
